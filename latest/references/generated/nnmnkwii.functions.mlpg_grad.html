nnmnkwii.functions.mlpg_grad¶

nnmnkwii.functions.mlpg_grad(mean_frames, variance_frames, windows, grad_output)[source]¶

MLPG gradient computation

Parameters are almost same as nnmnkwii.functions.mlpg(). See the function docmenent for what the parameters mean.

Parameters:
  • mean_frames (numpy.ndarray) – Means.
  • variance_frames (numpy.ndarray) – Variances.
  • windows (list) – Windows.
  • grad_output – Backpropagated output gradient, shape (T x static_dim)
Returns:

Gradients to be back propagated, shape: (T x D)
Return type:

numpy.ndarray
